ReadMe Generator

This project uses command-line application to generate a readMe file for user. This allows the project creator to devote more time to working on the project by answering a serious curated curated fields to create a proffessional high-quality readMe. When the user inputs their responces the logic will automatically auto populated a pre-determined template coded within the javascript file. Once the file has been created, the user can utilize the same code base to update the readME in real time. If the user runs the comman-line application again, the readME will automatically override with new information. | GitHub Repo
